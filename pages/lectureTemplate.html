SQL Query Example

Creating and querying database tables:

-- Create a students table
CREATE TABLE students (
  student_id INT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(255) UNIQUE,
  enrollment_date DATE DEFAULT CURRENT_DATE
);
-- Query with JOIN
SELECT s.name, c.title
FROM students s
INNER JOIN enrollments e ON s.student_id = e.student_id
INNER JOIN courses c ON e.course_id = c.course_id
WHERE c.credits >= 3;

Normalization in Practice

❌ Denormalized

Order_ID | Customer_Name | Items
1001 | John Doe | Book, Pen, Paper
1002 | Jane Smith | Laptop, Mouse

Problems: Repeating groups, update anomalies

✅ Normalized

Orders: Order_ID | Customer_ID
Order_Items: Order_ID | Item_ID | Quantity
Items: Item_ID | Item_Name | Price

Benefits: No redundancy, easier updates
